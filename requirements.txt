# AI Dialogue TUI — Зависимости проекта

Версии пакетов зафиксированы для обеспечения стабильности и безопасности.

---

## Основные зависимости

| Пакет | Версия | Назначение |
|-------|--------|------------|
| `textual` | 0.49.0 | TUI-фреймворк для создания интерфейса командной строки |
| `aiohttp` | 3.9.1 | Асинхронный HTTP-клиент для работы с Ollama API |

---

## Установка

```bash
pip install -r requirements.txt
```

---
